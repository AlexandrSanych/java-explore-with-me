# Explore With Me

Приложение для публикации и поиска событий с возможностью модерации.

### Дополнительная функциональность
- **Moderation Enhancement** — расширенная модерация событий: публикация, отклонение,
- отправка на доработку с комментарием, история модерации.

### Технологии
- Java 21, Spring Boot 3.3.2, PostgreSQL, Docker, Maven
